# VidMark 更新日志

## [2026-05-13] 产品规格与 MVP 路线

- [决策]: 产品名确定为 VidMark，内部可简称 VD，但公开展示统一使用 VidMark。
- [决策]: VidMark 定位为 Think Anytime 的视频精读子系统，而不是独立孤岛项目。
- [决策]: MVP 优先打通 YouTube，X 视频支持放到同一架构下的后续阶段。
- [进展]: 明确了视频捕获、字幕获取、中文精译、高能片段、阅读器 UI、笔记、Obsidian 写入和本地 Bridge 八个模块。
- [进展]: 制定了十阶段 MVP 更新计划，每个阶段都要求独立提交并推送到 GitHub。

## [2026-05-13] 字幕数据模型

- [进展]: 新增 VidMark 视频、字幕、片段和笔记数据模型。
- [进展]: 新增时间戳格式化、字幕排序、当前字幕查找和片段归一化测试。
- [踩坑]: 共享包构建默认会编译测试文件，已与 extension 包保持一致，排除 `*.test.ts`。

## [2026-05-13] YouTube 页面识别

- [进展]: 新增 YouTube watch URL 和 youtu.be 短链接识别。
- [进展]: 输出统一 VidMark 视频元数据，为浏览器入口做准备。

- [进展]: 新增 YouTube caption track 元数据解析。
- [进展]: 新增 timedtext XML 到 VidMark 字幕句的归一化能力。

## [2026-05-13] 中文精译接口

- [进展]: 新增 VidMark 字幕翻译请求和响应类型。
- [进展]: Bridge 新增 VidMark 翻译 prompt 构建和 JSON 输出解析。
- [进展]: 扩展 API 客户端新增 `/api/vidmark/translate` 调用。
- [踩坑]: Bridge 构建配置已排除 `*.test.ts`，避免测试文件进入运行时输出。

## [2026-05-13] VidMark 阅读器面板

- [进展]: 新增阅读器状态模型，支持字幕、高能、笔记三个视图。
- [进展]: 新增 React 阅读器骨架，支持关闭、字幕空状态、笔记输入和时间戳笔记列表。
- [进展]: VidMark 入口已从静态 HTML 面板升级为可扩展的阅读器组件。
- [踩坑]: React 阅读器进入 content bundle 后体积明显增大，MVP 阶段先接受，后续可评估懒加载。

## [2026-05-13] 高能片段

- [进展]: 新增 VidMark 高能片段请求和响应类型。
- [进展]: Bridge 新增高能片段 prompt 和 JSON 解析逻辑。
- [进展]: 片段边界会归一化到字幕句起止时间，并忽略无法匹配字幕的片段。
- [进展]: 阅读器“高能”视图已支持渲染片段列表和点击跳转。

## [2026-05-13] Obsidian 视频卡

- [进展]: 新增 VidMark 视频卡保存请求和响应类型。
- [进展]: 新增视频卡 Markdown 生成逻辑，包含来源、高能片段、双语摘录和用户笔记。

- [进展]: Bridge 和扩展 API 已接入 `/api/vidmark/save-card`。

## [2026-05-13] YouTube 字幕接入阅读器

- [进展]: 新增从 YouTube 页面脚本提取 `ytInitialPlayerResponse` 的能力。
- [进展]: VidMark 入口打开后会尝试读取 YouTube caption track，并把字幕句注入阅读器。
- [进展]: 阅读器会在字幕异步加载后保留当前状态并刷新字幕列表。

## [2026-05-13] 翻译、高能和保存闭环

- [进展]: VidMark 读取 YouTube 字幕后会通过 Bridge 自动请求中文精译。
- [进展]: 精译完成后会继续请求高能片段，并把片段列表注入阅读器。
- [进展]: 阅读器新增“保存”按钮，可把当前视频、字幕、片段和笔记写入 `50-VIDMARK/`。
- [边界]: 如果本地 Bridge 未启动或 Codex 不可用，阅读器仍保留原始字幕，不中断基础观看流程。

## [2026-05-13] MVP 文档与收尾验证

- [进展]: README 新增 VidMark 使用说明、运行要求和当前边界。
- [验证]: 完整运行 `npm run check` 和 `npm run build`。
- [边界]: MVP 已打通 YouTube 字幕读取、中文精译、高能片段和 Obsidian 视频卡；X 视频和无字幕 Whisper 转写仍为后续阶段。
