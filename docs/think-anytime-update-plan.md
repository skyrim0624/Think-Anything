# Think Anytime 更新计划

更新日期：2026-05-11

## 产品目标

Think Anytime 的目标是成为一个在 Chrome 浏览途中随时可用的阅读思考层：用户看到任何文本、图片、视频画面或网页内容时，可以快速提问、追问、发散，并把真正有价值的内容沉淀进 Obsidian，之后再被 Agent 主动想起和调用。

## 本轮原则

- 先强化“随时思考 + 可再次调用”的闭环，不急着堆新入口。
- 每一步完成后必须运行检查和构建，再提交并推送。
- 对外命名统一为 `Think Anytime`；历史内部包名 `twyr` 暂不强行迁移。
- 截图素材库、完整视频多帧理解、拖拽文件等能力先作为后续路线，不进入本轮硬目标。
